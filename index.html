Examples

Example 1

Pagination With Ajax Load

C#

@using MVCPagination.Components
@{
    var modelPaginning = new Pagination.ModelPaginning();
    modelPaginning.Id = "mvc-pagination";
    modelPaginning.TotalPages = 50;
    modelPaginning.StartPage =  Convert.ToInt32(!String.IsNullOrEmpty(Request["page"]) ? Request["page"].ToString() : "1");
    modelPaginning.VisiblePages = 5;
    modelPaginning.First = "Primeiro";
    modelPaginning.Prev = "Anterior";
    modelPaginning.Next = "Próximo";
    modelPaginning.Last = "Último";
    modelPaginning.Href = "?page={{pagenumber}}";
    modelPaginning.ShowControl = true;
    modelPaginning.ShowPrevNext = true;
    modelPaginning.ShowFirstLast = true;
    modelPaginning.HrefVariable = "{{pagenumber}}";
}

@using (Html.MVCPagination(modelPaginning)){ }

Js

    <script>
    $('.mvc-pagination').mvcPagination({
        totalPages: @Html.Raw(modelPaginning.TotalPages),
        visiblePages: @Html.Raw(modelPaginning.VisiblePages),
        href: "@Html.Raw(modelPaginning.Href)",
        startPage: @Html.Raw(modelPaginning.StartPage),
        first: "@Html.Raw(modelPaginning.First)",
        prev: "@Html.Raw(modelPaginning.Prev)",
        next: "@Html.Raw(modelPaginning.Next)",
        last: "@Html.Raw(modelPaginning.Last)",
        showPrevNext: @Html.Raw(modelPaginning.ShowPrevNext.ToString().ToLower()),
        showFirstLast: @Html.Raw(modelPaginning.ShowFirstLast.ToString().ToLower()),
        showControl: @Html.Raw(modelPaginning.ShowControl.ToString().ToLower()),
        hrefVariable: "@Html.Raw(modelPaginning.HrefVariable)",
        onClick: function (event, page) {
            $.ajax({ url: "test.html?page="+ page, cache: false }).done(function(html) {  $( "#results" ).append(html);});
        }
    });
    </script>

Available Options

The following list describes what options you have for the option object:

  • Id: Id of the pagination container (a DOM element)

  • TotalPages: The number of items per page. The maximum number of pages is calculated by dividing the number of items by items_per_page (rounded up, minimum 1). Please note: This value is only for calculating the number of pages. The actual selection of the items correlating to the current page and number of items must be done by your code in your callback function! Default: 0

  • StartPage: The page that is selected when the pagination is initialized. Default: 1

  • VisiblePages: Maximum number of pagination links that are visible. Default: 5

  • First: Text for the "First"-link that go to first page number. Default: First

  • Last: Text for the "Last"-link that go to last page number. Default: First

  • Prev**: Text for the "Previous"-link that decreases the current page number by 1. Default: Previous

  • Next: Text for the "Next"-link that increases the current page number by 1. Default: Next

  • **Href: Link target of the pagination links.

  • ShowControl: If this is set to false, the Controls "Previous|Next" and "First|Last" don´t is show. Default: true

  • ShowPrevNext: If this is set to false, the "Previous|Next"-link is don´t is show. Default: true

  • ShowFirstLast: If this is set to false, the "First|Last"-link is don´t is show. Default: true

  • HrefVariable: Variable to definition of the page number. Default {{number}}

Available Class Config

  • paginationClass: The root style for pagination component (default: 'pagination'). You can use this option to apply your own style

  • nextClass: The style for next link (default: 'next'). You can use this option to apply your own style

  • prevClass: The style for prev link (default: 'prev'). You can use this option to apply your own style

  • lastClass: The style for last link (default: 'last'). You can use this option to apply your own style

  • firstClass: The style for first link (default: 'last'). You can use this option to apply your own style

  • pageClass: The style for page link (default: 'page'). You can use this option to apply your own style

  • activeClass: The style for actual page link (default: 'active'). You can use this option to apply your own style

  • disabledClass: The style for disabled link (default: 'disabled'). You can use this option to apply your own style

Available Events

  • onClick: A callback function that is called when a user clicks on a pagination link. The function receives two parameters: the new page index and the pagination container (a DOM element). If the callback returns false, the event propagation is stopped. Default value: function(){return false;}. This callback function is essential for the functionality of the pagination! It should contain code that updates your content.
